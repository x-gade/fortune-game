# AGENTS.md

## Описание проекта

Fortune Game представляет собой desktop-приложение для проведения викторин в формате Что Где Когда с визуальным колесом выбора вопросов.

Приложение предназначено для:

* офлайн-мероприятий
* трансляций через OBS
* интерактивного управления игрой в реальном времени

Ключевая особенность архитектуры заключается в разделении:

* окна управления
* окна отображения для участников

---

## Цели проекта

### Основные

* визуализация колеса вопросов
* управление игрой через административный интерфейс
* отображение игрового процесса на отдельном экране
* таймер для каждого вопроса
* поддержка видео-вопросов
* гибкая конфигурация через JSON

### Перспективные

* редактор раундов и команд через UI
* сохранение игровых сессий
* экспорт результатов
* расширенные визуальные эффекты

---

## Архитектура

Проект построен по слоевой модели:

```
UI → Controller → Services → Models → Data
```

### Принципы

* UI не содержит бизнес-логики
* Controller управляет состоянием и координацией
* Services реализуют бизнес-логику
* Models описывают структуры данных
* Data слой отвечает за хранение

---

## Основные компоненты

### Controller

#### ui/app.py

Центральный управляющий компонент.

Функции:

* хранение текущего состояния игры
* управление игровым циклом
* обработка пользовательских действий
* координация между окнами
* управление таймером

Основные поля:

* current_question
* current_team_id
* remaining_seconds

Основные методы:

* select_round
* spin_for_team
* start_timer
* pause_timer
* stop_timer
* mark_correct
* mark_wrong
* reset_current_question
* reset_round_questions
* reset_all_questions
* add_question

## UI слой

### ui/admin_window.py

Окно управления игрой.

Функции:

* выбор раунда и команды
* запуск колеса
* управление таймером
* обработка результата ответа
* управление вопросами:

  * список вопросов

  * изменение статуса used
  * сброс вопросов
  * добавление новых вопросов

Публичное окно.

Отображает:

* видео

Компонент визуализации колеса.

---

### ui/timer_widget.py

Компонент отображения таймера.

### ui/scoreboard_widget.py

Компонент отображения счета команд.

## Игровой цикл

1. выбор раунда
2. выбор команды
3. запуск колеса
4. определение вопроса
5. отображение вопроса
6. воспроизведение видео при необходимости
7. запуск таймера
8. получение ответа
9. фиксация результата
10. обновление счета
11. пометка вопроса как использованного
